# Cycling Company Website

A modern, responsive website for a cycling company built with Jekyll and hosted on GitHub Pages.

## Quick Start

```bash
# Install dependencies
bundle install

# Run locally
bundle exec jekyll serve

# Build for production
bundle exec jekyll build
```

Visit `http://localhost:4000` to view the site locally.

## Setup Instructions

See [SETUP_GUIDE.md](SETUP_GUIDE.md) for detailed setup and deployment instructions, including:

## Project Structure

- `_posts/` - Blog posts and news
- `_layouts/` - Page templates
- `_includes/` - Reusable components
- `assets/` - Stylesheets, scripts, images
- `_config.yml` - Main configuration
- `index.markdown` - Homepage
- `about.markdown` - About page

## Built With

- [Jekyll](https://jekyllrb.com/) - Static site generator
- [Minima](https://github.com/jekyll/minima) - Theme
- [GitHub Pages](https://pages.github.com/) - Hosting
